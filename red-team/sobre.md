# Sobre

O <mark style="background-color:red;">Red Team</mark> é um teste de seg info avançado, simulando ataques reais de APTs para conseguir avaliar as defesas de uma organização.\
\
A principal diferença de um red team ops para o pentest tradicional é o tempo.&#x20;

Utilização de OSINT para recon de ambiente.\
Criação de spear phishing em whales (executivos de alto nível).\
Criação de backdoor específico para bypass de EDR/AV.\
Após foothold, planejar persistencia, escalação de privilegios e movimentação lateral.\
Planejar como será feita a extração de arquivos do ambiente.\
Pensar em bypass para DLP, utilizando outros procolos para envio de arquivos como ICMP, DNS.\
\
<mark style="background-color:red;">Red team</mark> só serve para melhorar os processos de monitoramento e detecção de ameaças do <mark style="background-color:blue;">Blue Team.</mark>\
\
