# Criptografia assimétrica

ECDSA (Elliptic Curve Digital Signature Algorithm)

É um sistema criptográfico que permite criar assinaturas digitais

* É utilizado no Bitcoin com a curva **secp256k1**.

\
RSA (Rivest–Shamir–Adleman)

É um algoritmo de criptografia assimétrica

* É utilizado em certificados SSL e PGP<br>
